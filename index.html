Code example

Simplicity check. We will divide the desired number x by all numbers from the range from two to x / 2 and look at the remainder of the division. If a divisor that divides the number x without a remainder is not found in this range, then we have a prime number.

            
const isPrime = (number) => {
  if (number < 2) {
    return false;
  }
  let divider = 2;
  while (divider <= number / 2) {
    if (number % divider === 0) {
      return false;
    }
    divider += 1;
  }
  return true;
}
